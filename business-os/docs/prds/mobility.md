# PRD — Mobility (Terapia Física)

## Objetivo

Plataforma de gestión de clínica de terapia física con calendario de terapeutas, fichas de pacientes, evaluaciones, captura de leads vía WhatsApp y 3 agentes IA (intake, follow-up, cierre) que reduzcan el trabajo administrativo del terapeuta a casi cero.

## Hitos

### Core clínico
- [x] Calendario de terapeutas
- [x] Pacientes y evaluaciones
- [ ] Plan de tratamiento estructurado por paciente
- [ ] Notas SOAP por sesión
- [ ] Recordatorios automáticos pre-cita

### WhatsApp Bot
- [ ] Captura de leads desde WhatsApp Business
- [ ] Calificación automática (intake agent)
- [ ] Agendamiento desde WhatsApp con confirmación
- [ ] Follow-up de paciente que no volvió (re-engagement)

### Agentes IA
- [ ] Intake agent: clasifica lesión y sugiere terapeuta
- [ ] Follow-up agent: pregunta evolución entre sesiones
- [ ] Cierre agent: detecta paciente listo para alta y sugiere upsell

### Operación
- [ ] Dashboard ocupación de terapeutas
- [ ] Reporte semanal de adherencia al tratamiento
- [ ] Onboarding para 1er terapeuta piloto
