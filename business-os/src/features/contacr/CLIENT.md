# ContaCR — Sistema Contable para Costa Rica

## Producto
SaaS mensual para contadores en Costa Rica. Cada contador maneja N empresas (PYMEs).

## MVP
- Import CSV de movimientos (ingresos/gastos)
- Dashboard con KPIs y gráficos
- Plan de cuentas estándar CR

## Stack
- Next.js 16 + Supabase (dentro de Business OS)
- Tablas: contacr_empresas, contacr_cuentas, contacr_movimientos, contacr_cuentas_template

## Roadmap post-validación
1. Libro diario con partida doble
2. Libro mayor
3. Balance de comprobación, Balance general, Estado de resultados
4. Reportes fiscales D-101 / D-104
5. Facturación electrónica Hacienda (futuro)
