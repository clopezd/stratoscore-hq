WhatsApp Business API

Instrucciones para activar confirmaciones automáticas y recordatorios por WhatsApp.

¿Qué se activa con esto?

Automatizaciones que se activan una vez configurado
  • Confirmación automática 24h antes: el sistema envía un WhatsApp al paciente pidiendo que confirme su cita. Si responde SÍ, se confirma en Huli. Si responde NO, se cancela automáticamente.
  • Recordatorio 2 horas antes: aviso automático con dirección y hora de la cita.
  • Re-agendamiento: pacientes que cancelan reciben un mensaje 48h después invitándolos a reagendar.
Costo: gratis para las primeras 1,000 conversaciones/mes. MedCare necesitaría ~200-300/mes.

1 Crear cuenta Meta Business

~5 minutos
  1. Abrir business.facebook.com
  2. Si ya tienen una cuenta Business de MedCare (por Facebook Ads o Instagram), pueden usarla. Si no, crear una nueva.
  3. Datos a ingresar:
    • Nombre del negocio: MedCare Centro Médico Especializado
    • Email: gerencia@rmacarecenter.com (o el email oficial)
Si ya manejan el Instagram @medcare_cr o hacen ads en Facebook, probablemente ya tienen esta cuenta.

2 Crear app en Meta Developers

~10 minutos
  1. Abrir developers.facebook.com/apps
  2. Click en "Crear app"
  3. Seleccionar tipo: "Business" (Empresa)
  4. Nombre de la app: MedCare WhatsApp
  5. Asociar con la cuenta Business de MedCare del paso anterior
  6. Click "Crear app"
  1. En el dashboard de la app, buscar la sección "Agregar productos"
  2. Buscar "WhatsApp" y click en "Configurar"
Aquí debería verse el panel de WhatsApp en el dashboard de la app

3 Configurar número de WhatsApp

~10 minutos

Elija una de estas dos opciones:

RECOMENDADO

Opción A: Usar el número actual (8368-2100)

El mismo número que ya usan los pacientes. Se migra a la API oficial de Meta.

  1. En WhatsApp > API Setup → click "Add phone number"
  2. Ingresar: +506 8368-2100
  3. Seleccionar verificación por SMS
  4. Ingresar el código recibido
Al migrar el número a la API, la app de WhatsApp Business en el celular dejará de funcionar para ese número. Todas las conversaciones se manejan via la API (nuestro sistema). Si necesitan seguir usando WhatsApp manualmente, elijan la Opción B.

Opción B: Usar un número nuevo

Comprar una SIM nueva y usar ese número solo para el bot automático. El 8368-2100 sigue funcionando manualmente.

  1. Conseguir un número de celular nuevo (cualquier operador CR)
  2. Registrarlo en la API igual que la Opción A
Costo: solo el de la SIM (~₡1,000). El número nuevo se usa exclusivamente para confirmaciones automáticas.

4 Obtener las credenciales

~5 minutos — Este es el paso más importante

Una vez registrado el número, ir a WhatsApp > API Setup en el dashboard de la app:

  1. Phone Number ID
    Aparece debajo del número registrado. Es un número largo, ejemplo: 123456789012345
  2. Access Token permanente
    Click en "Generate access token". Seleccionar permisos: whatsapp_business_messaging y whatsapp_business_management.
    Copiar el token inmediatamente. Es largo (200+ caracteres). No usar el token temporal de 24h — necesitamos el permanente.

~1 minuto

Enviar estos 2 datos por un canal seguro (WhatsApp o email directo, NO en un grupo):

Access Token: _________________________________
Con estos datos, nosotros configuramos todo. Las confirmaciones automáticas se activan al día siguiente.
