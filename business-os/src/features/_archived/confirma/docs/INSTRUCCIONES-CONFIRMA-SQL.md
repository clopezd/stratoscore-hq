## 📝 Instrucciones

### 1. Abrir Supabase
https://supabase.com/dashboard/project/csiiulvqzkgijxbgdqcv

### 2. SQL Editor
- Click "SQL Editor" (menú lateral)
- Click "+ New query"

### 3. Copiar SQL del archivo
```
business-os/EJECUTAR_EN_SUPABASE.sql
```

### 4. Ejecutar
- Pega todo el SQL
- Click "Run" (Cmd/Ctrl + Enter)

### 5. Verificar
```sql
SELECT * FROM public.clients;
```
Deberías ver 4 clientes.
