# Website 3D — Landing cinematográfica

Crea una landing page inmersiva con animaciones 3D scroll-driven usando React Three Fiber y GSAP.

## Cuándo usar

- "Quiero una landing impresionante con 3D"
- "Necesito una web tipo Apple / Lusion"
- "Agrega un hero con modelos 3D"

## Qué genera

- Escena R3F con `Canvas` optimizado
- Scroll-driven camera animation (GSAP + ScrollTrigger)
- Modelo GLB de ejemplo (`public/models/`)
- Lighting, shadows, postprocessing
- Responsive con fallback 2D en móvil
- Loader con progreso

## Stack

- Three.js
- @react-three/fiber
- @react-three/drei (helpers)
- @react-three/postprocessing
- GSAP + ScrollTrigger
- Tailwind para UI overlay
